# Student Record Manager

A Python-based Student Record Management System.

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Email Validation using Regex
- File Handling
- Exception Handling

## Technologies Used

- Python
- Regular Expressions (Regex)
- File Handling
- Exception Handling

## How to Run

```bash
python student_manager.py
```
